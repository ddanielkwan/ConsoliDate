# ConsoliDate



## About

ConsoliDate is an application being developed by Peter Poydenko, Shernan Javier, Naftal Ted, and Ricky Kong.

## Installation

Use the package manager [pipenv](https://pypi.org/project/pipenv/) to install dependencies.

```bash
pipenv install
pipenv shell
flask run
```
